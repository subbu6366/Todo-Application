\# Todo Application



A responsive \*\*Todo Application\*\* built using \*\*HTML, CSS, and JavaScript\*\* that helps users create, manage, and organize their daily tasks. The application allows users to add, complete, delete, and save tasks using the browser's Local Storage, ensuring data persists even after refreshing the page.



\## Features



\- Add new tasks

\- Mark tasks as completed

\- Delete unwanted tasks

\- Save tasks using Local Storage

\- Automatically load saved tasks when the application opens

\- Responsive and clean user interface

\- User-friendly design



\## Technologies Used



\- HTML5

\- CSS3

\- JavaScript (ES6)

\- Bootstrap 4

\## How It Works



\### Add a Task

\- Enter a task in the input field.

\- Click the \*\*Add\*\* button.

\- The task will be added to the task list.



\### Complete a Task

\- Click the checkbox beside a task.

\- The task is marked with a line-through style to indicate completion.



\### Delete a Task

\- Click the trash icon beside a task.

\- The selected task is removed from the list.



\### Save Tasks

\- Click the \*\*Save\*\* button.

\- All current tasks are stored in the browser's Local Storage.



\### Load Tasks

\- When the application opens, previously saved tasks are automatically loaded from Local Storage.
